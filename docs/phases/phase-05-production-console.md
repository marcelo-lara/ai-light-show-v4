# Phase 5: Production Console

## Goal

Upgrade the frontend from a playback prototype into a review and generation console.

## Included Epics

- [Epic 08: Preview Console](../epics/08-preview-console.md)
- [Epic 09: Render Diagnostics](../epics/09-render-diagnostics.md)

## Deliverables

- Preset browser.
- Timeline display.
- Render job progress.
- Frame inspector.
- A/B compare.
- Parameter editor generated from preset schema.
- Fullscreen preview mode.

## Exit Criteria

- A user can pick a song, generate a show, review the result, tune parameters, regenerate, and approve an output without touching files manually.
