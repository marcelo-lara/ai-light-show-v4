# Phase 6: Quality, Performance, And Packaging

## Goal

Make the system reliable enough for repeated production use.

## Included Epics

- [Epic 12: Render Diagnostics](../epics/12-render-diagnostics.md)
- [Epic 11: Fixture Mapping And Export](../epics/11-fixture-mapping-and-export.md)

## Deliverables

- Visual regression tests.
- Browser-driven visual regression case inventory with named baseline captures.
- Performance budgets.
- Cache cleanup tools.
- Export validation.
- Fixture mapping support.
- Deployment and recovery docs.

## Exit Criteria

- Render time, memory use, cache size, and output validity are observable.
- Exported shows can be mapped to target hardware or downstream playback systems.
